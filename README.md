⭕ Tic Tac Toe ❌
The Ultimate Battle of X's and O's
"Simple to learn, impossible to master."

This is a clean, responsive, and interactive Tic Tac Toe game built using HTML, CSS, and Vanilla JavaScript. It focuses on crisp user feedback and efficient win-check algorithms.

📖 About the Project
This project is a modern web implementation of the classic paper-and-pencil game. The goal was to create a digital version that feels tactile and responsive, featuring a polished UI and a robust game engine that handles turn-based logic seamlessly.

🧠 Why I Built This
Algorithm Design

Developing a function to check for 8 possible winning combinations (rows, columns, and diagonals).

State Management

Handling "X" and "O" turns while preventing players from overriding occupied cells.

UI/UX Practice

Creating smooth hover effects and clear visual indicators for the winner.

🎮 Features & Mechanics
⚔️ Two-Player Mode
Classic local multiplayer functionality where players take turns on the same device.

🏆 Win/Draw Detection
The game instantly recognizes a winning line or a stalemate (draw) and highlights the winning sequence.

🔄 Instant Reset
A dedicated "New Game" button allows for quick restarts without refreshing the page.
