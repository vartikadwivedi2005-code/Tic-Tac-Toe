⭕ Tic Tac Toe ❌
The Ultimate Battle of X's and O's
"Simple to learn, impossible to master."

This is a clean, responsive, and interactive Tic Tac Toe game built using HTML, CSS, and Vanilla JavaScript. It focuses on crisp user feedback and efficient win-check algorithms.

📖 About the Project
This project is a modern web implementation of the classic paper-and-pencil game. The goal was to create a digital version that feels tactile and responsive, featuring a polished UI and a robust game engine that handles turn-based logic seamlessly.

🧠 Why I Built This
1) Algorithm Design

2) Developing a function to check for 8 possible winning combinations (rows, columns, and diagonals).

3) State Management

4) Handling "X" and "O" turns while preventing players from overriding occupied cells.

5) UI/UX Practice

6) Creating smooth hover effects and clear visual indicators for the winner.
